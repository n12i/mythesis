\section{常時微動測定および自由振動実験}
破壊実験とともに常時微動測定および自由振動実験も行なった。ここではそれら
について測定時期および測定方法などを述べる。

\subsection{常時微動測定}
振動の計測には速度計を用い、水平2方向および鉛直方向について3チャンネルあ
るいは4チャンネル同時測定をした。各サイトに置ける速度計の配置を以下に挙
げる。
\begin{enumerate}
 \item \七条邸
  \begin{itemize}
   \item 破壊実験前
	 \begin{description}
	  \item[測定日] 1995年6月15日
	  \item[測定チャンネル数] 3チャンネル
	  \item[速度計の配置] 地表面(1ch)、2階床(2ch)、小屋梁(3ch)(図--
	  \ref{bido3})
	  \item[方向] 水平2方向と鉛直方向
	 \end{description}
   \item 破壊実験直前
	 \begin{description}
	  \item[測定日] 1995年6月20日
	  \item[測定チャンネル数] 4チャンネル
	  \item[速度計の配置] 2階床西側、2階床東側、小屋梁東側、小屋梁西
	       側(図--\ref{fv})
	  \item[方向] 桁行方向(破壊実験の際加力する方向)
	 \end{description}
   \item 破壊実験後
  \end{itemize}

\end{enumerate}
